##  About The Project

**Kentab Insurance Agency** is a modern, full-stack web application designed to digitize the insurance brokerage process in Kenya. 

Traditionally, getting an insurance quote involves phone calls and manual paperwork. This platform solves that by offering a **seamless digital experience** where clients can explore products, generate real-time quotes, and initiate policies instantly.

**Key Objectives:**
* **Digitization:** Moving manual quote workflows online.
* **Speed:** Built on Next.js 14 for sub-second page loads.
* **Trust:** A professional, accessible UI that builds user confidence.

---

##  Key Features

* ** Comprehensive Product Catalog:** Dynamic pages for Motor, Health, Business, Travel, and Pet insurance.
* ** Smart Quote Engine:** Interactive multi-step forms that validate user input using **Zod** before submission.
* ** Live Chat Integration:** Real-time customer support widget.
* ** Automated Notifications:** Instant email alerts for quotes and inquiries via **Nodemailer**.
* ** Mobile-First Design:** Fully responsive layouts optimized for the Kenyan mobile market.
* ** SEO Optimized:** Server-side rendering (SSR) for maximum search engine visibility.

##  Getting Started

To run this project locally:

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/Caleb-Kiune/kentab.git](https://github.com/Caleb-Kiune/kentab.git)
    cd kentab
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Set up Environment Variables**
    Create a `.env.local` file:
    ```env
    GMAIL_USER=your-email@gmail.com
    GMAIL_APP_PASSWORD=your-app-password
    CONTACT_TO_EMAIL=admin@kentab.com
    ```

4.  **Run Development Server**
    ```bash
    npm run dev
    ```
